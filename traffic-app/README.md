## GeoTraffic Insights

Interactive traffic ETL explorer built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting started

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` – start the local dev server
- `npm run build` – production build
- `npm run preview` – preview the production build
- `npm run lint` – run eslint
- `npm run test` – run unit tests

## Deployment

Any static host that supports Vite builds will work (e.g., Vercel, Netlify, Cloudflare Pages, S3+CloudFront). Build with `npm run build` and deploy the `dist/` folder.
